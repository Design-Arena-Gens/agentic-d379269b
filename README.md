# भवन योजना (Ground Floor Layout)

यह परियोजना एक स्थिर वेब पेज है जो 24' × 12' केंद्रीय हॉल, दोनों ओर 12' × 11' के दो-दो कमरों और हॉल के दक्षिण-पश्चिम कोने में सीढ़ियों के साथ ग्राउंड फ्लोर लेआउट प्रदर्शित करता है। पेज पर इंटरैक्टिव फ़्लोर प्लान, फीट ↔ मीटर इकाई टॉगल और आवश्यक नोट्स शामिल हैं।

## पूर्वापेक्षाएँ

- आधुनिक वेब ब्राउज़र (Chrome, Edge, Firefox, Safari आदि)
- वैकल्पिक: किसी भी स्थैतिक फ़ाइल सर्वर (उदा. `npx serve`) यदि आप लोकली होस्ट करना चाहते हैं

## फ़ाइल संरचना

```
├── index.html    # मुख्य पेज
├── styles.css    # स्टाइलशीट
└── script.js     # इंटरैक्शन और इकाई टॉगल
```

## लोकली देखने के तरीके

1. इस रिपॉजिटरी को किसी फ़ोल्डर में रखें।
2. सीधे `index.html` को ब्राउज़र में खोलें।

   **या**

   ```bash
   npx serve .
   ```

   फिर ब्राउज़र में `http://localhost:3000` खोलें।

## प्रमुख विशेषताएँ

- SVG आधारित फ़्लोर प्लान जिसमें कमरों और हॉल के सटीक अनुपात दर्शाए गए हैं।
- फीट से मीटर में त्वरित इकाई टॉगल।
- कमरे के क्षेत्र पर होवर/टैप करने पर टूलटिप में नाम और माप।
- हिंदी में विवरण और नोट्स, 11.6 फुट ऊँचाई से संबंधित सुझाव।

## अनुकूलन

- कमरों के आयाम बदलने के लिए `index.html` के SVG `rect` तत्वों को संशोधित करें।
- अपने ब्रांडिंग के अनुरूप टाइपोग्राफी या रंग-पैलेट बदलने के लिए `styles.css` संपादित करें।
- अतिरिक्त इंटरैक्शन जोड़ने के लिए `script.js` में लॉजिक बढ़ाएँ।
